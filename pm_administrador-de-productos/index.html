Sistema de Adminstración de inventario de Productos

Inicio Categorias Productos

Descripción General

  • El sistema esta pensado para la administración de un inventario de productos.
  • El sistema hace uso del localStorage del navegador.

Administración de categorías

  • Descripción del formulario.
    Campo Tipo Validación
    Código Input-text Maximo 10 caracteres
    Descripción Input-text Maximo 50 caracteres
  • Los campos de código y descripcion son obligatorios.
  • El sistema no permite el registro de categorías existentes.
  • Boton "Guardar": guarda la categoria que se esta creando.
  • Boton "Listar": muestra en una tabla todas las categorias que se han creado.
  • Se puede eliminar una fila seleccionandola y presionando la tecla 'suprimir'.
  • El sistema no permite eliminar una categoria si esta ya esta relacionada con un producto.

Administración de Productos

  • Descripción del formulario.
    Campo Tipo Validación
    Codigo Input-text Maximo 10 caracteres
    Descripción Input-text Maximo 50 caracteres
    Categoria Select Minimo 4 opciones
    Cantidad Input-numero Solo numeros
    Precio Input-numero Solo numeros
  • En el formulario de registro de productos se puede relacionar el producto con las categorías creadas previamente en un campo select.
  • El sistema no se permite el registro de productos con el mismo código o con la misma descripción.
  • Boton "Guardar": guarda el producto que se esta creando.
  • Boton "Listar": muestra en una tabla todas los productos que se han creado.
  • Boton "Editar": muestra un formulario previamente llenado con los datos del producto que se piensa editar.
  • Boton "Borrar": borra un producto.
  • Boton "Borrar Todo": borra todos los productos.
